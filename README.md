# vue-mdEditor一个自己用的Markdown网络笔记本

> 一个基于Vue.js的Markdown编辑器，UI基于ElementUI，数据存储在LeanCloud
 
## Demo

[Demo](http://mdDemo.adamin.win)

email: a@qq.com
password :111

## 背景
之前一直在使用CmdMarkdown编辑器，而且非常好用，但是对于图片的上传必须升级会员才行，这个。。。
伴着学习Vue，所以就自己试着写一个吧(∩_∩) ，要求不高，能用就行~~

### 为什么用LeanCloud
因为免费，API简单，自己使用足够用~
